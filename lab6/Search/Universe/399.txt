If the page can be re-written into an encyclopedic article, please do so and remove this message. This is an incomplete list of flag nicknames, organized in alphabetical order by flag name. Almost no flag has any really official name, even if some of these unofficial names may be so widely accepted that they are virtually universally used anyway.
